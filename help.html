How to Use Clark College Events

Welcome to the Clark College Events extension! Below are some common questions and answers to help you get the most out of it.

1. What does this extension do?

It shows "Today," "Earlier Today," and "Tomorrow" events from Clark College calendars. Click the down arrow next to any event in the list to expand and view more details.

2. How do I filter events on the main page?

On the Home page, use the "Filter events..." input box to quickly narrow down today's and tomorrow's events. As you type, the extension will filter events by title, time, date, or description in real-time. This helps you find specific events without leaving the main page.

3. How do I search for upcoming events?

Click "üîç Search Events" in the navigation bar. Once you've typed at least three characters, matching future events (up to 60 days ahead) will appear under Events and Training Events. Click the down arrow next to any result to expand and view full details.

4. What if I don't see any events?

If no matching events exist for your query, you'll see "No events found." If today or tomorrow truly has no scheduled items, you'll see "No scheduled events."

5. How is "Earlier Today" determined?

"Earlier Today" includes events whose start time was more than one hour ago (Pacific Time). If an event began over an hour ago (and is not an "All Day" item), it appears under Earlier Today.
