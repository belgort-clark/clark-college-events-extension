Version 1.4.8/1.4.9

  • Improved loading experience: loading overlay now only appears if data loading takes longer than 2 seconds
  • Enhanced error handling with professional styled error messages that include a warning icon
  • Better timeout management: global 10-second timeout for all RSS feeds with graceful error display
  • Error state UI: event sections are properly hidden when feeds cannot be loaded
  • Increased padding around error messages for better visual presentation
  • Robust error recovery: extension continues to function properly even when network issues occur
  • Logo refinements: improved vertical alignment, balanced spacing, refined size, and softer drop shadow effect

  • Accessibility: navigation uses ARIA roles and aria-current, role="main" added, and popups now trap focus and close with Escape key
  • Popup formatting: removed unnecessary <br> tags after </p> in event descriptions
